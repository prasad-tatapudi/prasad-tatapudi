<br/>

Real-time face detection, recognition, and emotion analysis system with an interactive Streamlit dashboard and automated attendance logging — built as a modular, production-style application rather than a single notebook script.

| Aspect | Detail |
|---|---|
| **Stack** | Python, OpenCV, Streamlit, DeepFace, NumPy, Pandas |
| **Scale** | Multi-module architecture (`detect.py`, `utils.py`, `app.py`) with independent, testable components |
| **Performance** | Real-time video frame processing with Haar Cascade classification |
| **Security** | Local-only face encoding storage, no external data transmission |
| **Impact** | Automates attendance logging to CSV; demonstrates end-to-end CV pipeline design with graceful dependency degradation |
| **Repository** | [github.com/prasad-tatapudi](https://github.com/prasad-tatapudi) |

Designed with a three-tab dashboard (Detection, Recognition, Attendance Logs) and built to degrade gracefully when optional heavy dependencies (like DeepFace) aren't available — prioritizing robustness and real-world usability over a fragile happy-path demo.
